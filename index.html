About the App

A professional attendance tracking system built with modern Android technologies that allows employees to clock in/out using location services and stores data securely in Firebase.

Key Features:

  • Secure Firebase Authentication
  • Real-time database updates
  • Background location tracking
  • Modern Jetpack Compose UI
  • Role-based access control
